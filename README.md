# A Simple game that I'm making just to learn more about SDL2 and C++

Always gonna be things to do in this game, I just don't know these things are

Pre-requisites
- SDL2
- SDL2_image
- SDL2_ttf
- GCC>=7
- CMake>=3.8
- Make


For compiling

```$ mkdir build
$ cd build
cmake ../
make
```

For running
```
$ ./Shootemup
```
